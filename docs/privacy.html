プライバシーポリシー

最終更新日: 2026-02-26

1. 収集する情報

当アプリは、以下の情報を利用・収集する場合があります。アプリの主な機能に必要な限りでのみ扱います。

  • 位置情報（端末の現在地） — マップやリマインダー機能のため
  • 端末情報（OS バージョン、デバイスモデル等） — 障害調査や改善のため
  • ユーザーが送信したお問い合わせ内容（メールアドレス等） — 問い合わせ対応のため

2. 利用目的

収集した情報は、アプリ機能の提供、改善、ユーザー対応、法令に基づく対応のために使用します。

3. 第三者提供

法令に基づく場合を除き、ユーザーの同意なしに第三者へ個人情報を販売または提供しません。ただし、次の外部サービスを利用します：

  • Google Maps API — 地図表示のため（Google の利用規約・プライバシーポリシーが適用されます）
  • Formspree（お問い合わせフォーム） — お問い合わせメールの配送

4. 保管期間

問い合わせデータ等は対応完了後、合理的な期間を経て削除または匿名化します。

5. ユーザーの権利

ユーザーは自身の情報の開示、訂正、削除を求めることができます。お問い合わせ先にご連絡ください。

6. お問い合わせ

プライバシーに関するお問い合わせは、当ページのお問い合わせフォーム、またはアプリ内で指定の連絡先をご利用ください。
